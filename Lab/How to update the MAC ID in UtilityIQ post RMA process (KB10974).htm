How to update the MAC ID in UtilityIQ post RMA process? (KB10974)
Products
Advanced Metering Manager (AMM)
Issue
This KB Article documents the procedure to update the MAC ID in UtlityIQ after RMA process.  The configurations should be updated before customer deploy the new device in the field.
Resolution
Step 1: Review the technical information

  • Review the MAC ID and related technical information of the new and replaced device.  For example:


Step 2: Export the Device File

  • Login to CustomerIQ and search by MAC ID which is the unique identifier provided in the table above.  Please ensure the drop down list is set to MAC Address when the search is triggered.


  • Click the Export Data button shown on upper right hand of device details screen to export the “old” device details. Click the Save button on the pop up window afterwards.


  • Right click on the Job hyperlink at the upper left corner and open the link (Export Details Page) in a new tab.


  • Click on the Latest Results link at the upper left corner of the Export Details Page.  A file will be exported for modification.  It may take up to five minutes for the export process especially for large customer. The Run Status will show Running initially.


  • Refresh the page by clicking the Latest Results link.  Once the file export process is completed, the page will update with the Export Files button.


  • Click on the Export Files button and a little icon will appear.  Right click on the icon to bring up a drop down menu.  Select Save Link As... and save this Old Device file by typing in the file name.  Please see the example below with "ABC RMA 33143 AP OLD 21EA" as the file name.

Note:  By default, the file is exported in tvs format.  In order to reimport the file properly, it should be saved as a tab delimited file after modification.


Step 3: Rename the fields in UtilityIQ

  • Change the “Old” device name, Serial Number, and Utility Device ID by appending "OLD" to each.  These fields are unique and they cannot match the same device information on the “New” import file.

    • Click on the Edit AP Details button on the bottom of the Device Details page.

  • Scroll back up to the top of the Device Details page and all the editable fields should now be open for modification. Add “OLD” to the end of the device name, Serial Number, and Utility Device ID.

  • Details Page after editing


  • Retire the old device so that it will never be used again.  Scroll down to the Device Details page and click on Retire this AP button.

  • Add a note in the device history with the RMA details. Click the Device History link on the upper right of the page and enter the note.


Step 4: Open the Device File in Excel

  • Open the old Device File in Excel

    • Start a new Excel program

    • Click the Data tab on top of the page.

    • Click the Form Text button

  • Change the file format to All Files (*.*)

  • Locate the file and open

  • Select Delimited as file type

  • Select Tab as the Delimiters

  • Scroll the bar all the way to the right and highlight all columns. Select Text as Column data format and click Finish button.


Step 5: Update the Device File

  • Modify the device file to become the new import file

    • Change Column F and G from Maintenance to New

  • Copy and paste the New device MAC ID into column U and V

    • For AP device RMAs, we need to copy and paste the WAN and NAN IP addresses that are provided. The NAN IPv6 address is under column X and the WAN IPv6 address is under column AY. If the device is a Relay then there is no IP address change.

    • Click the File tab

    • Select Save As option and save the file as Text (tab delimited) format. We recommend to name it similar as the export file.  Please see the example below:


Step 6: Import the Device File

  • Import the file

    • Bring up UtilityIQ, click the Devices tab and click the Device Admin tab

  • Confirm the following when importing a device file:

    • The environment is correct

    • The Device Type is correct

  • Select the correct Import Type

  • Enter the Local File location

  • Click the Import Devices button

  • It may take one to ten minutes to finish the import process depending on the size of the customer. Once the import is complete, the following message will show up.

 

Step 7: Verify after import

  • Search for the New MAC ID and it will appear in the UtilityIQ.
